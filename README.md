# SkillVault — Premium Template Marketplace

Browse marketplace → buy with Stripe → download ZIP → creator dashboard.

## Stack
- Next.js App Router + TypeScript
- Supabase (PostgreSQL + Auth + Storage)
- Stripe Checkout (one-time payments)

## MVP
- [ ] Browse/search marketplace
- [ ] Template detail with preview + reviews
- [ ] Stripe purchase → download ZIP
- [ ] Creator upload and manage
- [ ] 5+ seed templates

## Getting Started
```bash
npm install
cp .env.example .env.local
npm run dev
```
